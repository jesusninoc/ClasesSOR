# Automatización y scripting
- Scripting
  - PowerShell: https://www.jesusninoc.com/11/02/curso-online-de-powershell-noviembre-2025/
  - Linux: crea un script de Bash que lea un archivo con una lista de nombres de usuario y use un bucle for para crearlos en el sistema con useradd.
- En Windows, usa el Programador de Tareas para crear una tarea que ejecute tu script de PowerShell de la sesión anterior todos los días a las 10 PM.
- En Linux, usa crontab -e para programar tu script de Bash para que se ejecute cada minuto (usando la sintaxis * * * * * /ruta/a/tu/script.sh).

# Virtualización avanzada
- Instala Docker Desktop en tu máquina. Abre una terminal y ejecuta el comando docker run -d -p 8080:80 httpd. Con este simple comando, habrás descargado y ejecutado un contenedor con un servidor web Apache. Abre tu navegador y visita http://localhost:8080 para ver la página de bienvenida.
